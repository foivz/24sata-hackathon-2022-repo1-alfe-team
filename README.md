- `react-query` - querys/mutations

## API

### Teams

- za dobit teamove usera -> api/teams/
- za dodat team -> POST : api/teams/ -> name
- za kupnje teama -> GET : api/spending/
- za dodat kupnje teams -> POST : api/spending/ -> amount, teamdId, itemId
